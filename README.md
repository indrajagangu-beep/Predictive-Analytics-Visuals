# 📈 Predictive Analytics Using Historical Data

## Overview

This project demonstrates how historical sales data can be analyzed to forecast future sales trends using **Power BI**. The dashboard helps businesses identify sales patterns, monitor key performance indicators (KPIs), and make data-driven decisions through predictive analytics.

---

## 🎯 Objectives

- Analyze historical sales data.
- Clean and preprocess the dataset.
- Visualize sales performance using Power BI.
- Forecast future sales using time-series forecasting.
- Generate business insights for decision-making.

---

## ✨ Features

- Interactive Power BI Dashboard
- Historical Sales Trend Analysis
- Sales Forecasting
- KPI Cards
- Monthly Sales Analysis
- Regional Sales Analysis
- Product Category Analysis
- Interactive Filters and Slicers

---

## 📂 Dataset Information

The dataset contains **365 historical sales records**.

| Column | Description |
|---------|-------------|
| Order Date | Date of the order |
| Product | Product name |
| Category | Product category |
| Region | Sales region |
| Sales | Sales amount |
| Profit | Profit earned |
| Quantity | Quantity sold |
